一个亿的项目正式启动了！

发表于 2017-09-09 | 分类于 开篇

开篇



前言

本人主要从事软件开发这块,现在主要是负责公司Android项目的开发，业余时间则喜欢看点书，运动和搞自媒体。前段时间利用晚上业余，又回味了一下吴军老师的《浪潮之巅》，然后准备睡觉的时候，躺床上辗转反侧死活睡不着。 于是就回忆了一下来深圳这几年的经历。且不说一把鼻涕一把泪，但还是挺心酸的。

从15年中旬来深圳，找到第一份实习工作(Android开发)。然后到年底换工作到现在的公司。期间做过兼职、接过外包私活、做过维护。对于一个大专生来说个人觉得还是很不容易的。我想这一点很多人应该深有体会的，好在我庆幸是这一行不是那么拼爹 拼背景 拼钱。要不对于一个三无人员真的只能蹲在墙角默默不敢说话了…。 搞技术的你必须的耐得住寂寞、受得了委屈、经得住诱惑还有就是加班 加班 加班

在技术这块现在的我主要是以Android为主。以及延伸(音视频/智能硬件/车载),平时喜欢倒腾javaWeb、服务器、C++这块。技术积累是有些。但我发现好多东西先前用过或者花了我大把时间研究过、解决过的问题随着时间一长发现还是有所忘记 至少不能完全把代码弄出来 只能记得主要步骤 具体的实现还是要去找下资料什么的。

原因在于:

1.没及时消化和总结

2.适当的做笔记

所以说不能指望东西纯靠记,尤其是搞技术的 每年技术都可能日新月异 及时你通宵了原理 但还是会发现有些重要的东西还是要记。所以这也是我想写点东西的原因之一吧。把它当成一个对生活/学习/技术总结的地方。把你总结的东西分享出去,可能帮助到一些有着同样问题的人。



所以我想

  • 记录分享生活/工作/学习中一些有意思的事
  • 磨炼耐心
  • 写给以后的自己看

把这个事当成一种生活态度
